Rules & Scoring
📐 Scoring Formula ▾
Score = (L + B) ÷ W
L = Load supported (max 15,000g)
B = 5,000g bonus if ALL 3 criteria met
W = Structure mass (measured to 0.01g)
✅ Tier 1 Compliance ▾
All 5 required for Tier 1. Tier 1 always ranks above Tier 2.
✓ Single structure — no detachable parts
✓ Wood + adhesive only
✓ Loading block 40–45cm from wall
✓ Loading block within 2.5cm of centerline
✓ Wall contact within 8cm width lines

Build Techniques
🏗️ Truss Patterns ▾
Warren
Alternating triangles, no verticals. Most common for cable-stay beams. Clean force paths.
Pratt
Verticals + diagonals toward center. Diagonals in tension — efficient for long spans.
K-Truss
Split diagonals at midpoint. Reduces buckling in tall compression sections near the wall.
Flat Panel
Minimal depth laminated flanges. Lightest option — must pair with cable-stay tension system.
🪵 Wood & Adhesives ▾
Light balsa (6–8 lb/ft³)
→ Tension members: cables, bottom chords, lateral bracing

Hard balsa (12–14 lb/ft³)
→ Compression members: top chords near wall
→ 15cm clearance creates sharp tower angle → extreme buckling risk → hard wood critical here

CA glue (cyanoacrylate)
→ Pin-apply for lightest possible joints
→ Use accelerator sparingly — brittle failure

Epoxy
→ High-stress compression nodes only — adds mass
